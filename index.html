Week 1
Click here and download the repository.
  • Import it into Eclipse like normal.
  • Once imported, open the Brain class and run the program. Make sure the window opens and you can see the robot.
  • The initialize method is like the setup method in Processing. It only runs once at the beginning of the program.
  • The loop method is like the draw method in Processing. It runs in a constant loop until the program ends.
  • Experiment with the following method to move the robot around.
    • driveDirect(int leftWheel, int rightWheel);
